# Saikumar Automation QEA Portfolio

A modern React + Vite portfolio website for Saikumar Naddunuri, showcasing QA automation engineering experience, skills, projects, and credentials.

## Overview

This project is a single-page portfolio built with:
- React
- Vite
- JavaScript/JSX

It includes:
- An animated terminal-style hero section
- Experience and achievement sections
- Skill and project highlights
- Contact and resume download options


## Prerequisites

Make sure you have the following installed:
- Node.js (recommended: 18 or above)
- npm

## Installation

From the project root, install dependencies:

```bash
npm install
```

## Run locally

Start the development server:

```bash
npm run dev
```

Then open the local URL shown in the terminal, typically:

```text
http://localhost:5173/
```

## Build for production

Create a production build:

```bash
npm run build
```

Preview the production build locally:

```bash
npm run preview
```

## Scripts

- `npm run dev` — starts the Vite development server
- `npm run build` — builds the app for production
- `npm run preview` — previews the production build locally

## Resume / Assets

The portfolio includes a downloadable resume PDF located in:

```text
public/Saikumar_Naddunuri_01072026.pdf
```

## Deployment

This app can be deployed to any static hosting platform such as:
- GitHub Pages
- Netlify
- Vercel
- Firebase Hosting

For Vite-based static deployments, build the project and upload the generated `dist/` folder.

## Notes

- The main portfolio content is implemented in `portfolio.jsx`.
- The React entry point is `src/main.jsx`.
- The app uses Vite’s React plugin configured in `vite.config.js`.
